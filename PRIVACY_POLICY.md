# Privacy Policy / Политика конфиденциальности

_Last updated: 16 Nov 2025_

## English

### 1. General Information
Hungarian Culture Exam ("the App") helps users prepare for the Hungarian Cultural Knowledge Exam. No account or registration is required. We do not automatically collect personal identifiers (name, email, address, advertising IDs).

### 2. Optional Feedback & User-Submitted Content
The App can allow users (in debug or feedback/report modes) to submit textual suggestions or reports. This content is sent to our backend and/or third-party services (e.g. Notion) solely to improve content quality. Please do not include personal information in feedback. Any personal data you voluntarily submit may be processed for support and improvement purposes.

### 3. Network Connections & External Services
The App connects to external services to provide educational content and features:
- Wikipedia / Wikimedia (fetching article summaries, images)
- LibreTranslate (translation of study material)
- Notion API (optional feedback reports)
- Our content API (if configured via environment variables)
- YouTube (opening or embedding educational videos)

Only necessary parameters (language code, subtopic identifiers, text to translate/report) are transmitted. We do not combine this data to create personal profiles.

### 4. Data Storage
Study progress, settings, and cached resources are stored locally using local databases (`hive`) and preferences (`shared_preferences`). Debug image reports may be stored temporarily in a local text file before batch upload (if debug mode is enabled).

### 5. Subscriptions
Currently, the App does not implement an auto-renewable subscription. If subscription features are added later, this section will be updated.

### 6. Children's Privacy
The App is intended for users aged 13+. We do not knowingly collect personal data from children. Feedback features should not be used to submit personal information. If you believe a child submitted personal data, contact us for deletion.

### 7. Permissions & Device Features
- Notifications: Used to schedule study reminders (with user consent). No personal data inside notifications.
- Sensors (accelerometer/gyroscope): Used for visual effects only; readings are not stored or transmitted.
- Network / WebView / External links: Used to display external educational resources (YouTube, Wikipedia, translation).

No biometric, contacts, or precise location data is collected.

### 8. Security
We use HTTPS for external requests when supported. There are no user accounts; environment-based API tokens are application-level only.

### 9. Data Sharing
We do not sell personal data. User-submitted feedback (if any) is shared only with the service necessary to receive/process it (e.g. Notion). No advertising networks are integrated.

### 10. Contact
Email: hungarycultureexam@gmail.com
For questions or removal of submitted feedback, contact us.

## Русский

### 1. Общая информация
Приложение Hungarian Culture Exam («Приложение») помогает пользователям подготовиться к экзамену по венгерской культуре. Регистрация или создание учётной записи не требуется. Мы не собираем автоматически личные идентификаторы (имя, email, адрес, рекламные ID).

### 2. Необязательная обратная связь и контент, отправленный пользователями
Приложение может позволять пользователям (в режиме отладки или обратной связи/отчётов) отправлять текстовые предложения или отчёты. Этот контент отправляется на наш сервер и/или сторонние сервисы (например, Notion) исключительно для улучшения качества контента. Пожалуйста, не включайте личную информацию в обратную связь. Любые личные данные, которые вы добровольно отправляете, могут обрабатываться для целей поддержки и улучшения.

### 3. Сетевые подключения и внешние сервисы
Приложение подключается к внешним сервисам для предоставления образовательного контента и функций:
- Wikipedia / Wikimedia (получение резюме статей, изображений)
- LibreTranslate (перевод учебных материалов)
- Notion API (необязательные отчёты обратной связи)
- Наш контент API (если настроен через переменные окружения)
- YouTube (открытие или встраивание образовательных видео)

Передаются только необходимые параметры (код языка, идентификаторы подтем, текст для перевода/отчёта). Мы не объединяем эти данные для создания личных профилей.

### 4. Хранение данных
Прогресс обучения, настройки и кэшированные ресурсы хранятся локально с использованием локальных баз данных (`hive`) и настроек (`shared_preferences`). Отчёты с изображениями в режиме отладки могут временно храниться в локальном текстовом файле перед пакетной загрузкой (если включён режим отладки).

### 5. Подписки
В настоящее время Приложение не реализует автоматически возобновляемую подписку. Если функции подписки будут добавлены позже, этот раздел будет обновлён.

### 6. Конфиденциальность детей
Приложение предназначено для пользователей от 13 лет и старше. Мы сознательно не собираем личные данные от детей. Функции обратной связи не должны использоваться для отправки личной информации. Если вы считаете, что ребёнок отправил личные данные, свяжитесь с нами для их удаления.

### 7. Разрешения и функции устройства
- Уведомления: Используются для планирования напоминаний об учёбе (с согласия пользователя). Личные данные в уведомлениях отсутствуют.
- Сенсоры (акселерометр/гироскоп): Используются только для визуальных эффектов; показания не хранятся и не передаются.
- Сеть / WebView / Внешние ссылки: Используются для отображения внешних образовательных ресурсов (YouTube, Wikipedia, перевод).

Биометрические данные, контакты или точное местоположение не собираются.

### 8. Безопасность
Мы используем HTTPS для внешних запросов, когда это поддерживается. Учётных записей пользователей нет; токены API на основе окружения являются только на уровне приложения.

### 9. Передача данных
Мы не продаём личные данные. Обратная связь, отправленная пользователями (если есть), передаётся только сервису, необходимому для её получения/обработки (например, Notion). Рекламные сети не интегрированы.

### 10. Контакты
Email: hungarycultureexam@gmail.com
По вопросам или удалению отправленной обратной связи свяжитесь с нами.
